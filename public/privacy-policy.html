Kebijakan Privasi - Jadwal Sholat

Terakhir diperbarui: 2024

Pengumpulan dan Penggunaan Data

Aplikasi Jadwal Sholat tidak mengumpulkan, menyimpan, atau membagikan data pribadi pengguna. Semua perhitungan dan fungsi aplikasi dilakukan secara lokal di perangkat Anda.

Izin yang Dibutuhkan

Aplikasi memerlukan izin berikut untuk berfungsi dengan baik:

  • Lokasi: Digunakan untuk menentukan koordinat pengguna guna menghitung arah kiblat yang akurat.
  • Sensor Perangkat: Digunakan untuk mengakses sensor kompas guna menentukan arah kiblat.

Data lokasi dan sensor hanya digunakan secara lokal dan tidak dikirim ke server manapun.

Penyimpanan Data

Aplikasi tidak menyimpan data pengguna secara permanen. Semua data yang diperlukan untuk operasi aplikasi disimpan sementara di perangkat Anda dan akan dihapus saat aplikasi ditutup.

Perubahan pada Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Kami akan memberitahu Anda tentang perubahan apa pun dengan memposting kebijakan privasi baru di halaman ini.

Hubungi Kami

Privacy Policy - Jadwal Sholat

Last updated: 2024

Data Collection and Usage

The Jadwal Sholat application does not collect, store, or share any user personal data. All calculations and application functions are performed locally on your device.

Required Permissions

The application requires the following permissions to function properly:

  • Location: Used to determine user coordinates for accurate Qibla direction calculation.
  • Device Sensors: Used to access the compass sensor for determining Qibla direction.

Location and sensor data are used locally only and are not sent to any server.

Data Storage

The application does not permanently store user data. All data required for application operation is stored temporarily on your device and will be deleted when the application is closed.

Changes to Privacy Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new privacy policy on this page.

Contact Us

If you have any questions about this privacy policy, please contact us at: radit@arkana.co.id
